# AI_Interview_Assistant
AI Interview Assistant is a web-based application developed using Python, Streamlit, and the Groq API to help users prepare effectively for job interviews. The system analyzes uploaded PDF resumes, extracts relevant information, and generates role-specific interview questions based on the candidate's profile. It also provides AI-generated answers and explanations, enabling users to practice and improve their interview performance. By leveraging Natural Language Processing (NLP), prompt engineering, and Large Language Models (LLMs), the application delivers personalized, interactive, and realistic interview preparation experiences through an intuitive user interface.
